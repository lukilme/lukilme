Projetos Pessoais

Big Data Analysis Cluster
Ver no GitHub, em constante melhora -> github.com/lukilme/big-data-analysis-cluster
Ambiente distribuído para análise de grandes volumes de dados, composto por Hadoop (HDFS + YARN/Tez), Spark(Scala e Python), Hive e Sqoop. Todo o cluster foi containerizado com Docker e orquestrado via Docker Compose e Ansible, visando simular um ecossistema de Big Data completo em ambiente local para fins educacionais e experimentação com fluxos de ETL, processamento em lote e integração com bancos relacionais.
Python Chess Logger
Ver no GitHub, em constante melhora -> github.com/lukilme/python-chess-logger
Aplicação Python que registra e analisa partidas de xadrez utilizando o motor Stockfish. A ferramenta interpreta arquivos PGN, avalia jogadas com base na força do motor e gera relatórios em formato estruturado (JSON/CSV), permitindo estudos posteriores de desempenho, identificação de imprecisões e recomendações de melhores movimentos. Ideal para jogadores que buscam aprendizado baseado em análise computacional.
